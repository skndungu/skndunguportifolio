The Recipe App is a simple web app where users can share their cooking recipes and experiences, either in terms of videos or images. Its also a very light-weight app, based on the fact that its built on Vue js

Being deveoped using the Nuxt js its a universal app, describes JavaScript code that can execute both on the client and the server side.

Technology

The recipe App is uses pure CSS3 to give for a mobile first layout. It also utilizes the Nuxtjs give a server side rendering first, Vue js is also used to create some components that aid in the page rendering.
